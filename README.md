## Brain Tumor Detection using Machine Learning

This project implements a brain tumor detection system using machine learning to identify brain tumors from MRI images.

## Problem Statement

Brain tumors are a serious medical condition that can affect people of all ages. Early detection and diagnosis are crucial for effective treatment and can significantly improve patient outcomes. Traditional methods for brain tumor detection often rely on manual analysis of MRI images by radiologists, which can be time-consuming, subjective, and prone to human error.

## Proposed Solution

We propose a machine learning-based approach to automate the process of brain tumor detection from MRI images. The system uses deep learning techniques, specifically Convolutional Neural Networks (CNNs), to analyze MRI scans and identify the presence of brain tumors. The model is trained on a large dataset of labeled MRI images, allowing it to learn the complex patterns and features associated with brain tumors.

## Dataset

The dataset used for this project consists of brain MRI images categorized into three classes:
1. **Glioma** - Malignant brain tumor originating from glial cells
2. **Pituitary** - Tumor in the pituitary gland at the base of the brain
3. **Normal** - Healthy brain scans with no tumors

The dataset is publicly available and contains thousands of MRI images, making it suitable for training robust deep learning models.
